Skelbimų Projektas

Paleisti programą reikalinga:
- Node.js (versija 18 ar naujesnė)
- npm

Projekto paleidimas
- npm install
- Įkelti "privateKey.json" failą į pagrindinę direktoriją
- npm run server

Pagal numatytąjį nustatymą, serveris veiks http://localhost:3000
